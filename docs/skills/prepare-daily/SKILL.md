# Prepare Daily — подготовка к дейлику

## Что делает скилл

Запускает `scripts/prepare_daily.py` и выдаёт тимлиду готовый брифинг:
1. Порядок выступлений (ротация)
2. Что каждый планировал вчера (для сверки)
3. Незакрытые action items из предыдущих дней
4. Повторяющиеся блокеры
5. Контекст последнего дейлика

## Использование

### Шаг 1: Определить команду и дату

- `team` — ОБЯЗАТЕЛЬНО. Спроси, если не указана.
- `date` — по умолчанию сегодня.

### Шаг 2: Запустить скрипт

```bash
python3 scripts/prepare_daily.py --team TEAM_NAME [--date YYYY-MM-DD] [--history N]
```

Параметр `--history` — сколько прошлых митингов сканировать (по умолчанию 7).

### Шаг 3: Показать результат

Выведи брифинг пользователю. Обрати внимание на:

- Просроченные action items (`due_date < сегодня`) — выдели отдельно
- Повторяющиеся блокеры — предложи вынести в отдельное обсуждение
- Если открытых items > 5 — предложи приоритизировать

## Зависимости

- `scripts/prepare_daily.py`
- Наличие `meetings/` с обработанными `structured.json`
